Apache Phoenix Server中文乱码问题解决

继上次解决Phoenix Server Header长度受限的问题之后，今天又发现了4.4版本的Phoenix Server对含有中文的查询并不友好（新出的东西还是慎用啊），虽然问题的解决方法比较简单，但是为了保持自己写文章的动力（第一次发现自己能把刷文章数辣么猥琐的事情说的这么清新脱俗），还是在这记录一下。

问题表现
